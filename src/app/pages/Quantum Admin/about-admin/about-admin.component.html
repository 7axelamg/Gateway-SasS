Preguntas Frecuentes

¿Tienes dudas? Tenemos respuestas para ayudarte con nuestra plataforma de análisis de datos.

¿Qué tipos de análisis de datos ofrece la plataforma?

Nuestra plataforma ofrece tres niveles de análisis: descriptivo (básico), diagnóstico (Power BI) y predictivo (Weka AI). Puedes empezar con el plan gratuito y escalar según tus necesidades, obteniendo desde reportes simples hasta modelos avanzados de machine learning.

¿Con qué fuentes de datos se puede integrar?

Compatible con múltiples fuentes: Excel, CSV, SQL Server, MySQL, MongoDB, APIs REST y servicios cloud como Google BigQuery y AWS Redshift. El plan Power BI incluye conectores premium para fuentes empresariales.

¿Cómo protegen mis datos?

Seguridad de nivel empresarial con: encriptación AES-256, autenticación multifactor, cumplimiento GDPR/LOPD, backups diarios y opción de almacenamiento on-premise en el plan Weka AI. Realizamos auditorías trimestrales de seguridad.

¿Puedo personalizar los dashboards y modelos?

Totalmente. El plan Power BI permite personalización ilimitada de dashboards. El plan Weka AI incluye desarrollo de modelos personalizados con nuestro equipo de data scientists, ajustados a tus KPIs específicos.

¿Qué soporte técnico incluye cada plan?

Básico: Soporte por email (48h respuesta)
Power BI: Soporte prioritario 24/5 + capacitación mensual
Weka AI: Soporte 24/7 con asignación de ingeniero dedicado + revisiones trimestrales

¿Hay período de prueba para los planes premium?

¡Sí! Ofrecemos 14 días gratis para probar todas las funciones del plan Power BI. Para Weka AI, realizamos una demostración personalizada con tus propios datos y un Proof of Concept sin compromiso. Contáctanos para programarla.
