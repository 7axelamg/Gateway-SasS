Preguntas Frecuentes

¿Tienes dudas? Tenemos respuestas para ayudarte con nuestra plataforma de gestión hotelera.

¿Qué características incluye el sistema de gestión?

Nuestra plataforma incluye control de habitaciones, gestión de reservas, facturación, reportes de ocupación y tarifas, check-in/out digital y herramientas para mejorar la experiencia del huésped.

¿Se integra con otros sistemas?

Sí, nuestro sistema se integra con los principales canales de distribución (Booking, Expedia), sistemas de pago, contabilidad y herramientas de marketing. También ofrecemos API para conexiones personalizadas.

¿Cómo protegen mis datos?

Seguridad de nivel empresarial con: encriptación AES-256, autenticación multifactor, cumplimiento GDPR/LOPD, backups diarios y opción de almacenamiento on-premise.

¿Puedo personalizar los dashboards?

Totalmente. Nuestra plataforma permite personalización ilimitada de dashboards para mostrar exactamente los KPIs que necesitas monitorear.

¿Qué soporte técnico incluye?

Ofrecemos soporte prioritario 24/5, capacitación mensual para tu equipo y revisiones trimestrales para optimizar el uso de la plataforma.

¿Hay período de prueba?

¡Sí! Ofrecemos 14 días gratis para probar todas las funciones de la plataforma sin compromiso.
